  \hypertarget{index_autotoc_md0}{}\doxysection{Commander}\label{index_autotoc_md0}
\mbox{\hyperlink{namespaceCommander}{Commander}} is a simple, extensible, and configurable command framework for C\# developers.

It\textquotesingle{}s based on a simple idea of giving string input to the program and returning a string containing the result of the execution of the command based on the input.\hypertarget{index_autotoc_md1}{}\doxysection{Features}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Automatic command naming based on method name (overridable by attribute).
\item Automatic conversion and piping of string input to relevant data type(s) based on the command\textquotesingle{}s method parameter.
\item Automatic usage example generation.
\item Automatic command documentation based on method attributes and the method itself.
\item Automatic input splitting based on whitespace, preserves single and double quoted strings.
\item Automatic command dispatching.
\item Includes a default help command.
\item Add custom parameter convertors.
\item Add custom input splitters.
\item Add custom help command.
\item Detailed command documentation with special character sequences.
\item Command groups and nestable commands.
\item Required and optional command parameters, with optional parameters having default value(s).
\item Organizational unit \textquotesingle{}Cog\textquotesingle{} which is a class that contains commands and it can optionally group commands together.
\item Dead simple execution through the program Run method.
\item Error handling.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{Installation}\label{index_autotoc_md2}

\mbox{\hyperlink{namespaceCommander}{Commander}} can be used to implement command systems for games, command line tools, chat bots etc.

It accomplishes this goal by enforcing the input of a command string to the program and forcing all commands to return a string containing the result of the commands\textquotesingle{} execution.\hypertarget{index_autotoc_md5}{}\doxysection{Quickstart}\label{index_autotoc_md5}
